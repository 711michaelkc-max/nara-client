# 나라장터 입찰 검색 앱 구현 계획

## 프로젝트 목표
나라장터(G2B)에 등록된 입찰 공고를 쉽고 빠르게 검색하고 확인할 수 있는 애플리케이션 개발.

## 🧑‍💻 개발 조언 및 아키텍처 제안 (User Review Required)

### 1. 플랫폼 및 기술 스택
- **웹 애플리케이션 (추천)**: 접근성이 좋고 개발 속도가 빠릅니다. 모바일 브라우저에서도 '앱처럼' 사용할 수 있도록 반응형 웹 또는 PWA(Progressive Web App)로 개발하는 것을 권장합니다.
- **추천 스택**: 
    - **Frontend**: Next.js (React 기반, SEO 및 성능 우수)
    - **Styling**: Tailwind CSS (빠르고 유연한 스타일링)
    - **통신**: Axios 또는 Fetch API
    - **배포**: Vercel (빠른 배포 및 테스트 용이)

### 2. 데이터 연동 (나라장터 API)
- **공공데이터포털 활용**: 나라장터 정보는 공공데이터포털(data.go.kr)의 오픈 API를 통해 가져올 수 있습니다.
- **필수 서비스 (3가지 신청 권장)**: 다음 3가지 서비스를 모두 신청해주세요.
    1.  **조달청_나라장터 입찰공고정보서비스**: (필수) 공고 검색, 투찰 마감일, 상세 정보 확인.
    2.  **조달청_나라장터 낙찰정보서비스**: (필수) 개찰 결과, 낙찰 업체, 1순위 금액 확인.
    3.  **조달청_나라장터 계약정보서비스**: (권장) 최종 계약 체결 내역, 기관/업체의 과거 실적 추적.
- **인증**: API 활용 신청 후 발급받은 `Service Key` (Decoding 된 키 필요)를 사용합니다. (하나의 키로 위 서비스 모두 사용 가능)

### 3. 기능 구현 가능성 (Feasibility Analysis)
사용자 요청 사항(1~8)에 대한 API 구현 가능성 분석 결과입니다.

| 번호 | 요구사항 | 구현 방안 (API/App) | 비고 |
| :-- | :--- | :--- | :--- |
| 1 | 키워드 검색 | **PC/Mobile App** | 공고명, 공고번호 기준 검색. |
| 2 | 상세 필터 | **App Logic** | **기간(날짜)**, **업무종류**(용역/물품), **예산범위** 필터 UI 구현. |
| 3 | 결과/낙찰 확인 | **API Mixed** | [입찰] 탭와 [개찰결과] 탭 분리, 낙찰자/유찰사유 확인. |
| 4 | 이력/저장 | **Local Storage** | 자주 찾는 검색 조건(키워드+필터) 저장 기능. |
| 5 | 시인성 개선 | **Design Rule** | **고대비 배색** (배경 흰색/검정 텍스트) 준수. 모호한 회색 사용 지양. |

### 4. 배포 및 서버 운영 계획 (Deployment Strategy)
사용자님의 질문("서버는 어떻게 운용하지?")에 대한 답변이자 추천 드리는 방식입니다.

**"직접 서버를 켜놓을 필요가 없는 클라우드 호스팅(Vercel)을 추천합니다."**

-   **개발 단계**: 사용자님 PC에서 개발하고 테스트합니다. (`localhost`)
-   **운영 단계**: 코드를 클라우드 플랫폼인 **Vercel**에 올립니다.
    -   **장점**:
        1.  **관리 불필요**: 서버 컴퓨터를 사거나 24시간 켜놓을 필요가 없습니다.
        2.  **무료**: 개인 프로젝트 규모에서는 평생 무료 등급으로 충분합니다.
        3.  **자동화**: 코드를 저장(Git Push)하면 자동으로 배포됩니다.
    -   **접속**: 클라우드에 올리면 `https://my-nara-app.vercel.app` 같은 인터넷 주소가 생겨 스마트폰으로 바로 접속 가능합니다.

### 5. 주요 기능 제안 (Confirming)
-   **스마트 검색 대시보드**: 자주 쓰는 키워드 원터치 검색.
-   **통합 상세 뷰어**: 입찰 공고와 결과(낙찰/유찰) 통합 표시.
-   **알림**: (추후 고도화) Vercel의 Cron Job 기능을 이용해 주기적으로 공고를 체크하고 메일/메신저 알림 가능.
